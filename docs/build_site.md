# 建站

安装 Hexo 完成后

	$ hexo init <floder>
	...
	$ cd <floder>
	$ yarn install
	...

新建完成后，指定文件夹的目录

	.
	├── _config.yml
	├── package.json
	├── scaffolds
	├── source
	|   ├── _drafts
	|   └── _posts
	└── themes

* _config.yml	网站的配置信息
* package.json	应用程序的信息
* scaffolds		模版文件夹
* source		资源文件夹是存放用户资源的地方
* themes		主题文件夹
